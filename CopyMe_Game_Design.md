# 🎨 CopyMe — Скопируй рисунок!

## Концепция

**Жанр:** Пиксель-арт головоломка / развивающая игра  
**Платформы:** iOS, Android (телефон + планшет)  
**Ориентация:** Вертикальная (portrait)  
**Аудитория:** Дети 4–12 лет (основная), подростки и взрослые (расширенная)  
**Суть:** Игрок видит оригинальный пиксель-арт и должен воспроизвести его на пустом холсте, пиксель за пикселем

| Элемент | Телефон | Планшет |

## 🎮 Базовая механика

### Рисование
- **Тап** — закрасить один пиксель выбранным цветом
- **Свайп** — провести линию пикселей (как карандаш)
- **Долгий тап** — пипетка (взять цвет с холста/оригинала)
- **Щипок** — зум для крупных сеток (32×32+)
- **Двойной тап на оригинале** — зум-лупа этой области

### Инструменты (нижняя панель)

| Иконка | Инструмент | Описание |
|--------|------------|----------|
| 🖌️ | Карандаш | Рисует по одному пикселю |
| 🪣 | Заливка | Заливает область одним цветом |
| ↩️ | Отмена | Отменяет последнее действие (до 50 шагов) |
| 🔍 | Лупа | Увеличивает участок для точной работы |
| 💡 | Подсказка | Показывает один неправильный/пустой пиксель |
| 📊 | Прогресс | Показывает % совпадения с оригиналом |
| 🧽 | Ластик | Стирает пиксель (возвращает в пустой) |

### Проверка результата
- **Кнопка «Готово!»** — сравнивает холст с оригиналом
- Неправильные пиксели подсвечиваются красной рамкой (мигают)
- Пустые пиксели подсвечиваются жёлтым
- Правильные — зелёная волна удовлетворения ✨
- Оценка: ⭐⭐⭐ (100%), ⭐⭐ (90%+), ⭐ (75%+)

## 🏆 Геймификация и фишки

### 1. 🌍 Карта приключений (основной прогресс)

Вместо скучного списка уровней — **карта мира** в пиксельном стиле:
- Каждая «страна» = тематический набор (Океан, Космос, Ферма, Город, Фэнтези...)
- По 10–15 уровней в наборе
- Боссы в конце каждого набора (сложный рисунок на время)
- Разблокируются последовательно, как в Candy Crush
- Мини-сюжет с персонажем-компаньоном (пиксельный котик/робот)

### 2. ⏱ Режимы прохождения

| Режим | Описание | Для кого |
|-------|----------|----------|
| 🧘 Дзен | Без таймера, без лимитов. Расслабься и рисуй | Малыши, медитация |
| ⏱ На время | Уложись в таймер — бонусные очки | Соревновательные дети |
| 🧠 По памяти | Оригинал показывают 10–30 сек, потом скрывают! | Развитие памяти |
| 🔢 По номерам | Каждый пиксель пронумерован, как раскраска | Самые маленькие |
| 🪞 Зеркало | Рисуешь только половину — вторая отражается! | Изучение симметрии |
| 🎲 Рандом | Случайный рисунок, случайная сложность | «Мне скучно!» |

### 3. 🎖 Достижения и награды

**Коллекция ачивок:**
- 🐣 «Первые шаги» — завершить первый уровень
- 🎯 «Снайпер» — 10 рисунков на 3 звезды подряд
- ⚡ «Молния» — уровень за менее чем 30 секунд
- 🧠 «Фотограф» — 5 уровней в режиме «По памяти»
- 🎨 «Палитра» — использовать все цвета палитры в одном рисунке
- 🔥 «На огне!» — 7 дней подряд без перерыва (серия)
- 🦉 «Перфекционист» — 0 ошибок в рисунке 32×32+
- 🏗 «Архитектор» — создать и опубликовать свой рисунок

**Ежедневные/еженедельные квесты:**
- «Нарисуй 3 рисунка сегодня»
- «Завершай уровень без подсказок»
- «Попробуй новый режим»

### 4. 👤 Аватар и кастомизация

- Пиксельный аватар игрока
- Разблокируемые элементы: шляпы, очки, фоны, рамки
- **Зарабатываются за звёзды**, не за деньги
- Аватар «растёт» с уровнем игрока (из цыплёнка в дракона)

### 5. 🏅 Дуэли и социалка

- **Дуэли** — два игрока рисуют один и тот же рисунок на время. Кто точнее и быстрее?
- **Галерея** — лучшие работы игроков (модерация!)
- **Челлендж другу** — отправь свой рисунок другу как вызов
- **Семейный режим** — два игрока на одном планшете (split screen)

### 6. 💫 Визуальные награды и соки

- ✨ Частицы при правильно поставленном пикселе
- 🎆 Фейерверк при завершении уровня на 3 звезды
- 🔊 Приятные звуки тапов (разные для разных цветов!)
- 📳 Лёгкая вибрация при ошибке
- 🌈 Радужная анимация при серии правильных пикселей
- 🎵 Фоновая адаптивная музыка (ускоряется с таймером)

## 🧩 Дополнительные игровые инструменты и режимы

### 🔧 Инструмент 1: «Пиксельная лаборатория» (Редактор)

Свободное творчество:
- Пустой холст любого размера (8×8 → 64×64)
- Полная палитра цветов + пипетка
- Слои (для продвинутых)
- **Экспорт** как PNG / GIF
- **Публикация** в галерею — другие игроки смогут копировать твой рисунок!
- Обучающий эффект: переход от копирования к созданию

### 🔧 Инструмент 2: «Анимашка»

Создание пиксельных анимаций:
- Покадровая анимация (2–8 кадров)
- Рисуешь каждый кадр как отдельный пиксель-арт
- Предпросмотр анимации в реальном времени
- Экспорт как GIF
- Обучающий эффект: основы анимации, понимание движения

### 🔧 Инструмент 3: «Пиксельный пазл»

Альтернативная механика:
- Готовый рисунок разрезан на кусочки (как пазл)
- Нужно собрать, перетаскивая куски на холст
- Вращение кусочков для дополнительной сложности
- Обучающий эффект: пространственное мышление, ротация

### 🔧 Инструмент 4: «Миксер цветов»

Мини-игра внутри игры:
- Три базовых цвета (красный, жёлтый, синий)
- Смешивай, чтобы получить нужный оттенок
- Используй полученные цвета в рисовании
- Теория цвета через практику
- Обучающий эффект: цветоведение, RGB/CMYK

### 🔧 Инструмент 5: «Загадочный контур»

Режим-загадка:
- Показан только контур (чёрные границы)
- Нужно угадать и раскрасить самостоятельно
- Несколько «правильных» вариантов принимаются
- Обучающий эффект: цветовое воображение, креативность

### 🔧 Инструмент 6: «Шифр пикселей»

Математический режим:
- Вместо цветов — математические примеры
- Реши пример → узнай какой цвет ставить
- `2+3=?` → ответ 5 → зелёный
- Адаптивная сложность (сложение → умножение → дроби)
- Обучающий эффект: математика + пиксель-арт

### 🔧 Инструмент 7: «Координатная сетка»

Обучение координатам:
- Каждый пиксель имеет координату (A3, B5...)
- Задания: «Закрась пиксель C4 красным»
- Постепенно → «Закрась (3,7) в RGB(255,0,128)»
- Обучающий эффект: координатная система, подготовка к программированию
